## Estrutura de Controle<br><br>
As estruturas de controle são responsáveis por definir o caminho que o programa deve seguir durante sua execução. Elas permitem tomar decisões e repetir ações com base em condições específicas.<br><br>

### Tipos principais:<br><br>

- **Seleção (Condicional):** executa diferentes blocos de código dependendo de uma condição. Exemplo: `SE...SENÃO`.
- **Repetição (Laços):** executa um bloco de código várias vezes enquanto uma condição for verdadeira. Exemplos: `ENQUANTO`, `PARA`.<br><br>

### Exemplo de estrutura condicional:<br><br>
**Verificação de idade:**
```
    SE (idade &ge; 18) ENTÃO
    ESCREVA "Maior de idade"
    SENÃO
    ESCREVA "Menor de idade"
    FIM SE
```
### Exemplo de estrutura de repetição:<br><br>
**Contagem de 1 a 5:**
```
    PARA i = 1 ATÉ 5 FAÇA
    ESCREVA i
    FIM PARA
```
<br>Essas estruturas são fundamentais para criar programas dinâmicos e inteligentes, que se adaptam a diferentes situações e dados de entrada.
