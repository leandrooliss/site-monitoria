## Algoritmos<br><br>
Um **algoritmo** é uma sequência finita de passos bem definidos que descrevem como resolver um problema ou realizar uma tarefa. Ele funciona como uma receita: se seguido corretamente, leva a um resultado esperado.<br><br>
### Características de um bom algoritmo:<br><br>
- **Não ambiguidade:** os passos não podem gerar interpretações diferentes.
- **Finitude:** deve terminar após um número limitado de etapas.
- **Eficiência:** deve usar bem os recursos disponíveis (tempo, memória).
- **Generalidade:** deve resolver um problema genérico, não apenas um caso específico.<br><br>

### Exemplos de algoritmos:<br><br>

- **Receita de bolo:** lista de ingredientes e passos para preparar.
- **Instruções para montar um móvel:** sequência de ações para chegar ao resultado final.
- **Fluxo de login em um sistema:** verificar usuário, senha e liberar acesso.<br><br>

### Formas de representar algoritmos:<br><br>

- **Descrição narrativa:** escrita em linguagem natural.
- **Fluxograma:** uso de símbolos gráficos para representar o fluxo lógico.
- **Pseudocódigo:** escrita estruturada com comandos semelhantes aos de programação.<br><br>


Os algoritmos são a base da programação. Antes de escrever um código, é essencial entender e planejar o algoritmo que resolverá o problema.
