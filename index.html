Sobre a Monitoria

Conteúdo da Disciplina

Estruturas de controle

Aprenda sobre condicionais, loops e estruturas de decisão.

Exemplo - Estrutura Condicional
SE (idade ≥ 18) ENTÃO
    ESCREVA "Maior de idade"
SENÃO
    Escreva "Menor de idade"
FIM SE

Algoritmos de Ordenação

Bubble Sort, Selection Sort, Insertion Sort e algoritmos mais avançados.

Exemplo - Bubble Sort:
PARA i = 0 ATÉ n-1 FAÇA
    PARA j = 0 ATÉ n-i-2 FAÇA
        SE vetor[j] > vetor[j+1] ENTÃO
            trocar(vetor[j], vetor[j+1])
        FIM SE
    FIM PARA
FIM PARA

Estrutura de Dados

Listas, pilhas, filas, árvores e grafos.

Exemplo - Pilha (Push):
PROCEDIMENTO empilhar(elemento)
    SE topo < TAMANHO_MAX ENTÃO
        topo = topo + 1
        pilha[topo] = elemento
    SENÃO
        ESCREVA "Pilha cheia"
    FIM SE
FIM PROCEDIMENTO
